# Data Janitor: System Restore 🎮

**A fast-paced arcade experience built in Unity.**

In this project, I combined my background in **Data Analysis** with **Game Design** to create a game where logic and agility meet.

## 🚀 Key Features & Implementation
* **Dynamic Scaling Difficulty:** Implemented C# logic to adjust game pace based on player performance.
* **Modular Architecture:** Developed clean, decoupled scripts for Player Controller, Spawning Systems, and Game Management.
* **Visual Communication:** Applied UI/UX principles to ensure a clear and engaging player feedback loop.

## 🛠️ Tech Stack
* **Engine:** Unity 2022+
* **Language:** C#
* **Art & UI:** Free Assets Pack
